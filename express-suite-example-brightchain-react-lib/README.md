# express-suite-example-brightchain-react-lib

React component library for digitaldefiance-express-suite-example-brightchain.

## Purpose

Reusable React components, hooks, and utilities shared across React applications.

## Structure

```
src/
├── components/     # UI components
├── hooks/          # Custom hooks
├── contexts/       # React contexts
└── utils/          # React utilities
```

## Usage

```typescript
import { Button, useAuth } from '@express-suite-example-brightchain/react-lib';
```
